  • ● Apparaît dès l'ouverture (≤500 ms)
  • ● Plein écran, impossible à manquer
  • ● Explique pourquoi (3 bénéfices clairs)
  • ● Étapes visibles d'office, plus à fouiller

Tradeoffs :
  • Plus intrusif : bloque l'écran à la 1ʳᵉ ouverture. Compensé par "Plus tard" qui dismiss 24h (vs 7j actuels — on revient plus vite si l'user a oublié).
  • Sur Android : on continue d'utiliser le natif beforeinstallprompt (la modal s'ouvre seulement sur iOS où il n'y a pas d'API native).
  • Détection install : si l'user clique "J'ai installé" mais ne l'a pas vraiment fait, on dismiss 30j. S'il rouvre depuis Safari il reverra dans 30 jours. Suffisant.
