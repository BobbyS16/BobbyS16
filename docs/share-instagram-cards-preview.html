Design system des cartes :
  • 📐 Format 1080×1920 (story Insta · 9:16 vertical)
  • 🎨 Background dark avec un accent coloré par type (gold/red/green/purple/blue)
  • 👤 Avatar + nom de l'user en haut à gauche (identité)
  • 🎯 1 KPI dominant en très grand au centre (Bebas Neue 90px) — accroche l'œil
  • 🔖 Tag de catégorie en haut ("NOUVEAU PR", "PRONO RÉUSSI"...) — context immédiat
  • 📍 Watermark Pacerank + URL en bas — viral driver (les viewers tap pour découvrir l'app)

Implémentation technique :
  • Génération de l'image en HTML Canvas côté client (pas de serveur, pas de coût)
  • Lib html2canvas ou code Canvas natif (~10 KB minifié)
  • Bouton "Partager" → navigator.share() avec le file → ouvre la sheet de partage iOS/Android
  • Fallback : download direct dans la galerie photo
  • L'user choisit ensuite Instagram Story dans la sheet → image arrive comme background
